# Hackathon Portal

## Description
Full-stack hackathon platform with frontend (React/Vite) and backend (Node.js).

## Features
- User registration
- Hackathon registration
- Challenges
- Leaderboard
- File uploads
- WebSocket real-time

## Tech Stack
- Frontend: React, Vite
- Backend: Node.js, Express
- Database: (models)
- Websocket

## Installation
Frontend:
cd frontend
npm install
npm run dev

Backend:
cd backend
npm install
npm start
